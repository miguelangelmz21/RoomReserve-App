# 🏨 Hotel Management System

A professional **Single Page Application (SPA)** designed to manage hotel room reservations efficiently. This project features a realistic 6-floor building architecture with categorized room types and a robust administrative control panel.

## 🎯 Project Objective
To develop a functional and responsive web application applying advanced concepts of rendering, routing, global state management, and UI design.

## 🛠️ Tech Stack
* **Environment:** [Vite] + [React 19]
* **Styling:** [Tailwind CSS] (Mobile First Approach)
* **Routing:** [React-Router-Dom]
* **State Management:** [Zustand]
* **Icons:** [Lucide-Icons]
* **Notifications:** [SweetAlert2]



## 🗺️ Navigation & Architecture
The application implements a secure routing system with 6+ distinct views:
* ` / ` : **Home** - Landing page and initial view.
* ` /login ` : **Login** - Secure user access.
* ` /dashboard ` : **Dashboard** - Guest view and room listing.
* ` /panel-admin ` : **Admin Panel** - Management and CRUD operations.
* ` /unauthorized ` : **Unauthorized** - Access control restriction.
* ` /404 ` : **Not Found** - Custom error handling.

## ✨ Key Features
* **Global State:** Efficient data sharing across components using **Zustand** to avoid prop drilling.
* **Responsive UI:** 100% Mobile-first design built entirely with **Tailwind CSS**.
